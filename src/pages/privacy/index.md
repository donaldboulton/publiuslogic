### Analytics Cookie Types

* __utma

ğŸ¤š This cookie is used to determine new and returning visitors.  It has an expiration time of 2 years.  If the ga.js library is executed and no _utma cookie exists, this will be recorded as the usersâ€™ first visit and a _utma cookie will be set. * __utmb

ğŸ¤š This cookie is used to determine a new session.  The cookie is set when the ga.js library executes and there is no _utmb cookie in place.  It has an expiration time of 30 minutes, therefore if a user is inactive for a period longer than this, a new cookie will be set when the library executes and the interaction will be recorded as a new session.

* __utmz

ğŸ¤š This cookie is used to determine the traffic source, medium, campaign name and campaign term which delivered the user to your website. This helps Google collect the data which can then help them to determine which traffic sources assist conversions within the multi-channel section of Analytics.

* __utmc

ğŸ¤š This Cookie waits 30 minutes, and then it expires. You see, __utmc has no way of knowing when a user closes their browser or leaves a website, so it waits 30 minutes for another pageview to happen, and if it doesnâ€™t, it expires.

* __utmv
